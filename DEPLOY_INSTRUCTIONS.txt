📝 STEP 2: Deploy Frontend on Vercel
------------------------------------
1. Go to: https://vercel.com
2. Sign up with GitHub
3. Click "New Project"
4. Import your GitHub repository
5. Framework Preset: "Vite"
6. Add Environment Variable:
   
   VITE_API_URL = YOUR_RAILWAY_URL_HERE
   
   (e.g., https://devbhoomi-backend.up.railway.app)
   
7. Click "Deploy"
8. Your frontend will be live! (e.g., https://devbhoomi-cyberthon.vercel.app)


📝 STEP 3: Update CORS on Railway
----------------------------------
1. Go back to Railway dashboard
2. Click on your backend service
3. Go to "Variables" tab
4. Add new variable:
   
   FRONTEND_URL = YOUR_VERCEL_URL_HERE
   
   (e.g., https://devbhoomi-cyberthon.vercel.app)
   
5. Service will automatically restart
